What it does

Plans before it acts

An orchestrator spawns specialised sub-agents (researcher, coder, writer, verifier) and runs them in parallel where possible.

You pick the model

Anthropic, OpenAI, Gemini, OpenRouter, Ollama, DeepSeek. Per role. Mix and match: orchestrator on Opus, summariser on Haiku.

Real safety, not vibes

Agent Shield scans every prompt, tool call, and output. Strict / balanced / permissive / off. F1 0.988 on prompt-injection.

Budget is mandatory

Per-run dollar cap, wall-clock cap, tool-call cap. The kernel refuses to bust the budget. No surprise $200 bills.

Anywhere on your terms

Desktop, Docker, VPS, or as a phone companion through nine channel adapters (Telegram, Discord, Slack, Signal, Matrix, Bluesky, Mastodon, email, voice).

Open source, no paid tier

MIT licensed. No telemetry. Your prompts go to whichever provider you configured, nowhere else.

Install

pipx install 'maverick-agent[installer]'
maverick init

PyPI name is maverick-agent. The [installer] extra pulls the wizard into the same pipx environment so maverick init resolves. Requires Python 3.10+.

$ pipx install 'maverick-agent[installer]'
installed package maverick-agent 0.1.1 / maverick-installer 0.1.1
$ maverick init
How do you want to set this up?
consumer Four questions, safe defaults. About a minute.
advanced Pick every model, channel, safety level, budget.

Who it's for

Everyone else

Consumer mode is four questions. The wizard fills the rest with safe defaults: strict safety, sandboxed shell, $5 budget cap, web search on.

  • Download, double-click, four questions
  • No terminal needed once installed
  • You can switch to advanced any time

Engineers + power users

Advanced mode exposes every knob: per-role model assignment, custom MCP servers, plugin allowlist, per-tool rate limits, retention, persona, notifications, full sandbox + channel matrix.

  • Plugin SDK with maverick plugin new
  • Programmatic REST API at /api/v1
  • MCP server in both directions
